## ğŸ¨ Customization Guide

### Changing Colors
Edit the CSS variables in `style.css`:
```css
--primary-color: #ff69b4;
```

```html

    <img src="Pics/new-profile.jpg" alt="Profile Picture">

    <p>Interests: Interest1, Interest2, Interest3</p>
```
